# 🧮 RaspberryGlace Counter

A simple and beginner-friendly **Counter Application** built with **HTML, CSS, and JavaScript**.  
This project is perfect for practicing DOM manipulation, event handling, and basic styling.

---
# Why "RaspberryGlace Counter"

because it's styled with a sweet and beautiful pink tone named as **Raspberry Glace Pink**

---
## ✨ Features

- **Increment (+)**: Increases the counter value by 1.
- **Decrement (-)**: Decreases the counter value by 1.
- **Reset (0)**: Resets the counter back to zero.
- **Responsive Design**: Clean and centered layout with CSS styling.
- **Real-time Updates**: The displayed number changes instantly when buttons are clicked.

---
## 📚 Learning Goals

- Practice **HTML structure** for building UI.
- Use **CSS styling** to create a neat and responsive layout.
- Apply **JavaScript event listeners** to add interactivity.
- Understand **DOM manipulation** by updating elements dynamically.
